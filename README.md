# Alex Carter - Professional Photography Portfolio Template

A high-performance, SEO-optimized photography portfolio template built with HTML5, Tailwind CSS, and lightweight JavaScript libraries.

- **SEO Optimized**: Includes Open Graph, Twitter Cards, Schema.org (JSON-LD), and semantic HTML.
- **Responsive Design**: Fully responsive layout optimized for all devices.
- **Fast Performance**: Optimized image loading, minimal blocking resources.
- **Modern Tech Stack**: Tailwind CSS via CDN (can be built via CLI), Swiper.js, GLightbox, AOS animations.
- **Structured Data**: `Person` and `CollectionPage` schema markup for better search visibility.

## üîç SEO Configuration

Before deploying, make sure to update the following files with your real information:

1.  **`index.html` & `portfolio.html`**:
    *   Update `<title>` and `<meta name="description">`.
    *   Update `yourdomain.com` placeholders in `<link rel="canonical">` and `og:url`.
    *   Update social links in the `Schema.org` script block.

2.  **`robots.txt`**:
    *   Update the `Sitemap:` URL to match your domain.

3.  **`sitemap.xml`**:
    *   Find and replace `https://www.yourdomain.com` with your actual domain URL.

4.  **`manifest.json`**:
    *   Update the app name and description.
